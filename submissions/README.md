# Submissions

This directory is where creator BApp submissions live.

Each subdirectory is one BApp:

```
submissions/
  your-bapp-slug/
    index.tsx        ← Your React component (required)
    manifest.json    ← BApp metadata (required)
    preview.png      ← Example output screenshot (optional)
```

See the [template](../template/) for starter files and the [README](../README.md) for full submission instructions.

---

> **Review process:** All PRs are reviewed before being added to the live BApp Store at [0rdinals.com/bapp-store](https://0rdinals.com/bapp-store). Approved BApps will be added in the next release.
